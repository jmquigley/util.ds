Class: Deque

Deque

A double ended Queue class

new Deque( [maxSize], cmp)

The deque can be unlimited in size (default) or set to a maxium size when constructed. When the max size is exceeded, then the front item is automatially removed from the queue and the new item is placed in the queue (dependin which type of insert is calld).

Parameters:
Name Type Argument Default Description
maxSize number <optional>
 0

The maximum size for this queue.
cmp function null

a comparator function used for searching within the container.
